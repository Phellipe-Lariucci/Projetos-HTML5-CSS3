Projeto de Android.

Projeto Redes Sociais.

Projeto HTML/CSS construído para treinar o posicionamento e imagens e iframes.

Saiba mais...

Aqui, 'simulei' o efeito visual de uma tela de smartphone, com um menu de links laterais que, ao ser clicado, muda a tela para a rede desejada e, no final, oferece um link para a mesma. Como o objetivo era utilizar os iframes e o posicionamento da imagem dentro deles, todas as redes são, na verdade, imagens estáticas e estarão desatualizadas.
